第七阶段：进阶选学

学习时长：按兴趣安排 · 学习目标：解锁更多玩法，让 AI 帮你做更复杂的事

前面 5 个阶段已经让你能"日常用 AI"。这一阶段是给有兴趣的同学准备的——不是必修，也没必要按顺序学。

建议挑一个你最感兴趣的玩，玩通透后再看下一个。每节后面我都标了入门、中等、较难，按自己节奏来。

一、上传文件：让 AI 读你的资料入门

这是最简单也最实用的进阶能力。AI 不仅能聊天，还能"读"你给的文档。

能读什么？

  • PDF（说明书、合同、论文、电子书）
  • Word / Excel / PPT
  • 图片（拍照、截图、报告单）
  • 网页链接（直接发 URL 让它读）

怎么用？

  1. 在 AI 输入框旁找 📎 / + / "上传" 按钮
  2. 选择本地文件或拍照
  3. 等几秒让 AI 读完，然后提问

实用场景示例

场景 1：读医院化验单
拍照上传 → "请用大白话解释这份化验单，标出指标异常的地方，并告诉我可能的原因和建议下一步做什么。"
场景 2：读购房/租房合同
上传 PDF → "请逐条解释这份合同，标出对我（买/租方）不利的条款，建议如何与对方协商修改。"
场景 3：读家电说明书
拍照上传 → "请告诉我这台洗衣机怎么设置'羽绒服洗涤'，并提醒我使用时的注意事项。"
场景 4：让 AI 读论文/电子书
Kimi 是这方面的强项，能读几十万字的长文档，问它"第三章主要讲什么"、"这本书最重要的 3 个观点"。
推荐工具：Kimi（读长文档最强）、豆包（拍照识别快）、WPS AI（直接在 Word/Excel 里用）。

二、AI 画图：用文字"画"出图入门

用文生图Text-to-Image。你用一段文字描述想要的画面，AI 给你画出来。背后是另一种大模型，叫"扩散模型"（Diffusion Model）。技术，AI 能根据你的描述生成图片，做朋友圈封面、PPT 配图、头像、海报都行。

主流 AI 画图工具

工具 归属 特点 新手友好度
即梦 字节 免费、中文好、有 App ⭐⭐⭐⭐⭐
可灵 快手 图、视频都能生 ⭐⭐⭐⭐
豆包 / Kimi 内置画图 — 无需另装 App ⭐⭐⭐⭐⭐
通义万相 阿里 免费、风格丰富 ⭐⭐⭐⭐
Midjourney 海外 画质天花板 ⭐⭐（付费+英文+特殊网络）

画图提示词公式

【主体】+【场景/环境】+【风格】+【画质/视角】+【细节要求】

例：

一只穿着宇航服的橘猫，漂浮在月球表面，背景是地球，皮克斯卡通风格，超精细，4K 高清，俯视视角

常用风格关键词

  • 真实摄影："超写实、单反摄影、4K"
  • 动漫："宫崎骏风格、二次元、新海诚风格"
  • 插画："扁平插画、水彩画、油画、国风水墨"
  • 3D："皮克斯风格、3D 渲染、C4D"
AI 生成图片中的文字、手指、表盘等细节经常出错——这是当前技术通病。重要场合用前一定要看清楚。

三、智能体 / Agent：把常用任务封装成"小机器人"中等

智能体英文 Agent，也叫"AI Agent""GPTs""Bot"。本质是：把一段你写好的提示词 + 一组工具能力固化下来，做成一个专门干某件事的"小 AI"。下次直接调用，不用重复交代背景。是把"提示词 + 知识 + 工具"打包，做成一个专门处理某类任务的小 AI。

普通对话 AI = 一个万能的客服，每次都得跟他重新介绍你的需求。
智能体 = 你训练好的私人助理，专做某件事，每次直接叫他干活。

常见智能体场景

  • 翻译官：固定翻译某门语言，按你常用风格
  • 简历优化师：上传简历自动给修改建议
  • 育儿百科：基于权威育儿资料回答问题
  • 菜谱助手：根据你说的食材推荐菜
  • 面试官：模拟岗位面试给你练手

怎么用？（不需要编程）

平台怎么找
豆包App 里的"发现"→"AI 智能体" / "智能体广场"
扣子（Coze）coze.cn ——字节出品，可视化创建智能体
Kimi+Kimi 网页版/App 的"应用广场"
ChatGPT GPTschatgpt.com（需海外账号）

自己做一个简单智能体的 4 步

  1. 打开扣子或豆包智能体创建页
  2. 给它一个名字和头像
  3. 填写"人设与指令"——也就是它的固定提示词（"你是一位 XX，你的任务是 XX，回答时要 XX"）
  4. 上传你希望它"学习"的资料（可选），保存发布
动手做一个"朋友圈文案大师"智能体：人设设定为"你是擅长写小红书爆款风格朋友圈的文案专家"，上传 10 条你喜欢的文案。之后每次需要发朋友圈，直接用这个智能体即可。

四、AI 视频生成中等

用文字或图片生成短视频，从几秒钟开始。

工具能做什么
即梦文字/图片生成 5-10 秒视频，免费
可灵视频质量较高，国内顶级
Sora（OpenAI）海外，效果惊艳，需特殊网络
Runway / Pika海外专业工具
适合：做短视频开头、制作产品介绍小动画、做创意素材。
不适合：长视频（目前都只能生成几秒到十几秒）、严格控制内容的场景。

五、AI 编程 / 自动化较难

即使你不会编程，AI 也能帮你做一些简单的自动化：

  • Excel 公式："帮我写一个 Excel 公式，把 A 列日期是周末的行标红"
  • 批量处理文本："请把下面这些电话号码格式统一成 138-xxxx-xxxx"
  • 简单脚本："帮我写一个 Python 脚本，把文件夹里所有 jpg 改名"

有兴趣深入的可以了解：

  • Cursor / Windsurf / Claude Code：AI 编程工具，能直接写代码、改代码
  • n8n / Dify / 扣子：可视化搭建 AI 工作流

六、本地 AI / 私有部署较难

本地 AI把开源大模型下载到自己电脑上运行，不联网也能用。优点是隐私安全、不花钱；缺点是需要好显卡，效果不如顶级在线 AI。就是把开源模型装到自己电脑上：

  • 优点：不联网、不上传数据、隐私安全
  • 缺点：需要好显卡、效果不如最新的在线 AI、安装复杂

主流工具：

  • Ollama —— 一键运行开源大模型
  • LM Studio —— 图形界面，最适合新手
本地 AI 对小白门槛较高（需要懂一点命令行、有合适的显卡），不建议作为入门方向。除非你有特殊隐私需求，否则用在线 AI 性价比更高。

七、API 调用 / 接入自己的应用较难

如果你或团队想把 AI 嵌入自家的网站、小程序、办公系统，需要通过APIApplication Programming Interface，应用程序接口。说白了就是"程序间的快递员"——你的程序通过 API 把问题发给 AI，AI 把答案传回来。调用。

常用 API：

  • OpenAI API（ChatGPT 背后）
  • Anthropic Claude API
  • 豆包 API、通义 API、DeepSeek API（国内）

这块属于开发者范畴，普通用户了解概念即可。
