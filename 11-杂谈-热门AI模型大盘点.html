五、3 个"反直觉"经验

1. 不要迷信"最新"
新版本不一定比老版本好——尤其是免费版上线一个月内常常不稳定。新模型出来先观察两周再换。

2. 不要只看一个榜单
LMArena 主观偏好高的模型可能在 SWE-bench真实软件工程能力评测——给模型一个 GitHub issue 让它修代码。比传统编程题更贴近实际工作。 上很弱；MMLU 高分的模型可能日常聊天很死板。把任务对应到 2~3 个相关榜单交叉看。

3. 国产模型 ≠ 弱
截至 2026 年中，DeepSeek、GLM、Kimi、Qwen 在多项榜单上已经追平甚至反超同期闭源前沿。差距主要在生态 / 工具链 / 多模态成熟度，不在"硬实力"。

六、决策建议（一句话总结）

📱 手机 / 日常聊天：豆包（中文最舒服）或 ChatGPT App

💻 电脑写代码：Claude Code + Claude Opus，或 cc-switch 切到 DeepSeek

📚 读长文档：Kimi 网页版（拖文件进去就行）

🌐 查最新信息：Perplexity 或带联网的豆包 / Kimi

💰 预算紧 / 跑量：DeepSeek API（按量计费便宜一个数量级）

🔒 注重隐私 / 内网：本地跑 Llama 4 或 Qwen 3.5（参考本地命令行工具）
