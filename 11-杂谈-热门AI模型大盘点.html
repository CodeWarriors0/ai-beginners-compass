热门 AI 模型大盘点选型参考

一张随时翻看的笔记：去哪查、谁能干什么、什么场景该挑谁。

数据快照：2026 年 5 月 · 模型迭代飞快，下次打开前先看一眼排行榜

新模型几乎每周都在发，名字一个比一个长（V4 Pro Max Reasoning Plus……）。这篇不是教程，而是一份"记到本子上随时翻"的参考——告诉你：

  1. 权威排行榜去哪看（不要听营销稿）
  2. 当前主流模型有哪些、各自擅长啥
  3. 遇到具体任务该挑谁
注意 本文写于 2026 年 5 月，模型名字、版本号、能力分数过几个月就会变。文中所有数据都给了原始链接，用之前去原网站再核对一下。

壹 · I看排行榜，认准这 5 个 GitHub 来源

下面这些是开源、可复现、有学术声誉的排行榜——它们不卖广告，数据相对客观。

项目它评什么用法
lm-sys / FastChat
★ 38k+		 Chatbot Arena现已更名为 LMArena。让两个匿名模型同时回答你的问题，你投票哪个更好，用国际象棋的 Elo 评分计算"人类偏好"。是目前最权威的"主观体验"榜单。 的开源代码与榜单数据 看人类主观偏好——哪个模型让真人觉得"答得更好"
CLUEbenchmark / SuperCLUE
★ 3k+		 中文综合能力评测（语言 / 专业 / Agent / 安全 四大维度） 看哪个模型最懂中文——国产模型在这里有主场优势
open-compass / OpenCompass
★ 5k+		 上海 AI 实验室出品，覆盖 100+ 数据集的综合评测框架 看客观题打分——数学、推理、代码各项分开看
EleutherAI / lm-evaluation-harness
★ 8k+		 HuggingFace Open LLM Leaderboard 背后的评测引擎 看开源模型的标准化分数——MMLU、HellaSwag 等
Hannibal046 / Awesome-LLM
★ 22k+		 "大模型大全"——按时间线收录所有重要模型与论文 想了解某个模型的来龙去脉时去翻
懒人路径 先看 lmarena.ai 看真人偏好排名,再去 SuperCLUEAI.com 验证中文表现,两者交叉印证基本不会踩坑。

伍 · V三个"反直觉"经验

一 · 不要迷信"最新" 新版本不一定比老版本好——尤其是免费版上线一个月内常常不稳定。新模型出来先观察两周再换。
二 · 不要只看一个榜单 LMArena 主观偏好高的模型可能在 SWE-bench真实软件工程能力评测——给模型一个 GitHub issue 让它修代码。比传统编程题更贴近实际工作。 上很弱；MMLU 高分的模型可能日常聊天很死板。把任务对应到 2~3 个相关榜单交叉看。
三 · 国产模型 ≠ 弱 截至 2026 年中，DeepSeek、GLM、Kimi、Qwen 在多项榜单上已经追平甚至反超同期闭源前沿。差距主要在生态 / 工具链 / 多模态成熟度，不在"硬实力"。

陆 · VI决策建议（一句话总结）

手机日常聊天 豆包（中文最舒服）或 ChatGPT App
电脑写代码 Claude Code + Claude Opus，或 cc-switch 切到 DeepSeek
读长文档 Kimi 网页版（拖文件进去就行）
查最新信息 Perplexity，或带联网的豆包 / Kimi
多模态（图 / 视频） Gemini 3，或国内豆包 / 即梦
预算紧 / 跑量 DeepSeek API（按量计费便宜一个数量级）
注重隐私 / 内网 本地跑 Llama 4 或 Qwen 3.5（参考本地命令行工具）
